# **Running MovieApi**

- Open terminal.
- Use `Makefile` for
    1. Set GOPATH .
    2. Install all dependencies.
    3. Build API.
    4. Run API.

#### 1. **Set GOPATH**
```
make set-path
```

#### 2. **Install all dependencies**
```
make go-get
```

#### 3. **Build API**
```
make go-build
```

#### 4. **Run API**
```
go-run
```
