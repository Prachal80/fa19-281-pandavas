# **Running USERAPI**

- Use `Makefile` for
    1. Set GOPATH .
    2. Download and Install Dependencies.
    3. Build.
    4. Run.

#### 1. **Set GOPATH**
```
make set-path
```

#### 2. **Install all dependencies**
```
make go-get
```

#### 3. **Build API**
```
make build
```

#### 4. **Run API**
```
make run
```
